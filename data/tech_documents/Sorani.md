# Sorani — 次世代IoTプラットフォーム

## 概要

Soraniは、製造業向けのクラウドネイティブIoTプラットフォームです。老朽化したSCADAや独自システムからの移行に強みを持ち、既存設備のデータを統合・可視化・分析する基盤を提供します。OPC-UA / Modbus / MQTTなど主要プロトコルに対応し、異なるメーカーの設備データを単一プラットフォームで統合できます。10年以上前のレガシーSCADAからの移行実績が多数あり、標準4ヶ月プロセスで移行を完了します。

## 主要機能

- **データ収集・統合**: OPC-UA / Modbus / MQTT など主要プロトコルに対応。異なるメーカーの設備データを単一プラットフォームで統合。エッジデバイスによるローカル処理とクラウド連携を両立
- **可視化・分析**: ノーコードでダッシュボードを構築。KPI管理・トレンド分析・アラート設定。過去データの長期保存（最大10年）とトレンド分析
- **SCADA移行サポート**: 既存SCADAからのデータ移行ツールを標準提供。移行期間中は新旧システムの並行稼働が可能。10年以上前のレガシーSCADAからの移行実績多数

## 適用ユースケース

- **三菱系製造業（類似案件）**: 15年前のSCADAが拡張限界。Soraniへ段階移行し、システム運用コスト40%削減、新ライン追加リードタイム 3ヶ月→2週間を実現
- **自動車部品メーカー（複数拠点）**: 拠点ごとにシステムがバラバラで全体可視化が不可能な状態から、全拠点データをSoraniで統合してリアルタイム監視を実現
- 複数拠点・複数メーカー設備を一元管理したいケース
- 3ヶ月での社内提案に向けた資料提供対応も可能

## 技術仕様

**対応プロトコル**: OPC-UA / Modbus / MQTT

**移行プロセス（標準4ヶ月）**

1. 現状調査・接続設計（1ヶ月）
2. パイロット導入・並行稼働（2ヶ月）
3. 本番切り替え・旧システム停止（1ヶ月）
4. 運用開始・定着支援（継続）

**その他**: エッジデバイスによるローカル処理対応、ノーコードダッシュボード構築、過去データ最大10年保存
