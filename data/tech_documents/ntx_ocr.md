# NTX-OCR - AI帳票デジタル化ソリューション

## 製品概要

NTX-OCRは、製造現場で使われる紙の帳票・手書き記録を高精度AIで自動的にデジタルデータへ変換するソリューションです。従来のOCRが苦手とした手書き文字・罫線のない帳票・劣化した紙面にも対応し、製造業特有の帳票（検査記録・作業日報・受払票など）を専門に学習した独自AIモデルを搭載しています。スマートフォン・スキャナ・複合機から取り込んだ画像を数秒でデータ化し、既存のERPや生産管理システムへ自動連携します。

## 主要機能

### AI帳票認識
- 手書き・印刷混在帳票の高精度認識（認識精度 平均98.3%）
- 非定型帳票への対応（レイアウト不定な帳票も自動学習）
- 表・チェックボックス・押印・捺印の認識
- 多言語対応（日本語・英語・中国語）

### 入力チャネル
- スマートフォンアプリによるその場でのカメラ撮影→送信
- 複合機・スキャナからの自動取り込み（FTPポーリング対応）
- メール添付ファイルの自動処理
- バッチ処理（大量帳票の一括変換）

### データ連携・出力
- JSON / CSV / Excel形式でのデータ出力
- REST APIによる外部システム連携
- SAP・Microsoft Dynamics・弥生シリーズとの公式コネクタ
- 変換結果の確認・修正UIによるヒューマンインザループ検証

### 学習・カスタマイズ
- 自社帳票の追加学習（50枚〜でファインチューニング対応）
- 認識結果フィードバックによる継続的精度向上
- 帳票テンプレート管理（最大1,000種類）

**山陽食品工業株式会社（食品製造）**
HACCP管理記録の紙帳票をデジタル化。監査時の記録提出対応時間が3日から4時間に短縮。食品安全マネジメントシステム（FSSC 22000）の審査でデジタル管理が高評価を受けた。
